----------README----------
Uygulamanın kodları Eclipse IDE ortamında yazılmıştır.
Java 18 kullanılmıştır.
Database için PostgreSQL kullanılmıştır.
Java ve Postgresql bağlantısı JDBC ile sağlanmıştır.
Fxml dosyalarında el ile değişiklik yapabilmek için SceneBuilder kullanılmıştır.
SQL.txt dosyaları sql tablolarını oluşturmak için kullanılmıştır.
-------------------------
Programın çalıştırılması için kodları ve gerekli yüklemeleri yaptıktan sonra çalıştırmaya hazır hale gelecektir.
